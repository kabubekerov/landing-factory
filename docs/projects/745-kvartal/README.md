## Working Rule
Если меняется логика проекта, нужно обновлять не только код страницы, но и тот документ, который отвечает за соответствующий слой знания. Если подходящего файла нет, это повод либо создать новый документ в проектной папке, либо зафиксировать исключение через `decisions.md`.
